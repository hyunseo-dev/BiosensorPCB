# BiosensorPCB

## Biosensor Impedance Analyzer
AD5933을 이용한 임피던스 아날라이저 코드입니다.

**MainPCB.ino** 파일은 **PuTTY, 아두이노 시리얼 모니터**를 사용할 때 보드에 업로드하여 사용합니다.
**MainPCBPython** 파일은 **파이썬 터미널**에서 결괏값을 확인할 때 보드에 업로드하여 사용합니다.

## AD5933 Library
'https://github.com/mjmeli/arduino-ad5933' 라이브러리의 미완 부분을 구현하여 사용하였습니다.
